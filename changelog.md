<details>

<details>
<summary><strong>Artworks: <code>arctic</code> image category</strong> — #10 ⇄ #11 (⊶ 5bd0c9fc) by <a href="https://github.com/svengreb" target="_blank">@svengreb</a></summary>

↠ A new `arctic` category has been created that imports some images from [the old `nord-docs` repository][5]. This mainly includes “decorations“ that can be used for various purposes.

</details>

<details>
<summary><strong>Static images: <code>elements/separators/palettes</code></strong> — #12 ⇄ #13 (⊶ 2462a594) by <a href="https://github.com/svengreb" target="_blank">@svengreb</a></summary>

↠ A new `elements/separators/palettes` category has been created that provides various format and layouts of the Nord color palette.

</details>

<details>
<summary><strong>Static images: new <code>elements/functional</code> artboard with <code>links</code> category</strong> — #14 ⇄ #15 (⊶ 35d6bf51) by <a href="https://github.com/svengreb" target="_blank">@svengreb</a></summary>

↠ A new `elements/functional` [artboard][6] has been created that initially provides the `links` category. It contains elements that can be used as element to assign a e.g. a HTML link.
